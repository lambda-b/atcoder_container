# AtCoder コード

## 使用上の注意
1. vscodeをdevcontainerを使用して開く
2. online-judge-toolsを使用してログインしておく

```sh
$ oj login -u <ユーザー名> -p <パスワード> "https://atcoder.jp/"
$ oj login --check "https://atcoder.jp/"
```
